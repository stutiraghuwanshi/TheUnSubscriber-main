# **App Name**: Subscription Savior

## Core Features:

- Subscription Entry: Allow users to manually input their subscription details including name, renewal date, and cost.
- Renewal Reminders: Send email or SMS reminders 3 days before a payment is due. Use a tool to intelligently determine the delivery method.
- Spending Visualization: Display total monthly and yearly spending via charts and graphs.
- Subscription Overview: Display all active subscriptions in a list or grid format.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to convey trust and reliability, contrasting nicely with a light background.
- Background color: Light gray (#F0F4F8), providing a clean and modern look.
- Accent color: Muted green (#81C784) for positive affirmations, like payment confirmations.
- Body and headline font: 'PT Sans', a humanist sans-serif, for a blend of modernity and approachability.
- Use simple, outline-style icons to represent different subscription types.
- Clean and intuitive layout with a clear hierarchy to make managing subscriptions straightforward.
- Subtle transitions and animations when adding or editing subscriptions to provide visual feedback.
